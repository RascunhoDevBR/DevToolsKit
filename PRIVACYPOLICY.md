Privacy Policy – Dev Tools Kit

Last updated: March 8, 2026

This Privacy Policy explains how Dev Tools Kit, developed by Rascunho, handles information when you use the software.

1. Overview

Dev Tools Kit is designed to respect user privacy. The software operates locally on the user's computer and does not collect personal data.

2. Information Accessed by the Software

In order to provide its features, the software may access certain technical information about the computer, including:

CPU usage

system performance statistics

operating system information

hardware resource usage

This information is used only for displaying system information within the application.

3. No Personal Data Collection

Dev Tools Kit does not collect personal information, including:

names

email addresses

physical addresses

location data

personal files

4. No Internet Data Transmission

The software:

does not send data to external servers

does not store user data remotely

does not track users

All processing occurs locally on the user's device.

5. Third-Party Services

Dev Tools Kit does not integrate with third-party services, analytics tools, or advertising networks.

6. Data Security

Because the software does not collect or transmit personal data, there is no external storage of user information.

7. Children's Privacy

The software is not intended for children under 11 years of age.

No personal data is knowingly collected from minors.

8. Changes to This Privacy Policy

This Privacy Policy may be updated periodically. The updated version will replace any previous version.
